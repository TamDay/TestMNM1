# Uploads Directory

This directory is used to store uploaded room images.

## Default Images

The database includes references to the following room images:
- room1.jpg through room8.jpg

For demo purposes, you can:
1. Add your own room images here
2. Or the system will use placeholder images from placeholder.com

## Image Requirements

- **Format**: JPG, PNG, GIF
- **Max size**: 5MB
- **Recommended dimensions**: 800x600px
